# 🧭 Career-Life Designer Agent (人生战略规划与选择问询陪伴智能体)

> **拒绝灌鸡汤，拒绝死板问卷，拒绝大模型乱分析纯凭感觉。这是一个结合了“张雪峰式毒舌”与“反脆弱策略”的多轮聊天 AI 导师。它通过冷酷的框架限制大模型胡说八道，死卡你的真实筹码，强迫你面对现实，并吐出真正能落地防身的人生搞钱/转型路线图。**

## 🎯 它能实现什么？（扒开高大上，看具体干货）

这个项目在技术上不画大饼，它没有人工智能自我意识，本质上是一个基于 **FastAPI + 联网搜索 + 极度严厉的结构化提示词（Prompt）** 的智能体核心。但它切切实实能帮你做到以下 4 件事：

### 1. 像张雪峰一样逼你交出“生存底牌”
一般的 AI 聊着聊着就顺着你走了，这个 Agent 在后台死死盯着你的记忆账本（`user_profile`）。你跟它说想转行、想考研，它会立刻踩刹车逼问你：*“学历什么专业？家里能不能供得起？大城市买得起房吗？”* **如果不把家庭经济和现实基本盘交代清楚，它会卡住流程，反复盘问，直到戳破你的认知幻觉。**

### 2. 强行分流精力的 80/20 杠铃布阵
它绝对不给你温水煮青蛙的平庸建议（比如劝你多加班、考个不高不低的职称）。它的底层逻辑强制帮你的生活分流：
* **80% 的精力（防守池）**：老老实实干能保住社保、能托底的硬技能或主业，用来保命。
* **20% 的业余时间（进攻池）**：死磕低成本、高爆发的时代红利（如出海、自媒体、AI垂直应用），用来翻盘。

### 3. 设计 3 套不吊死在一棵树上的平行人生
拒绝单一路径焦虑。AI 会在后台算法中强行为你推演 3 条同时存在的 5 年人生版本：
* `版本 A（稳健搞钱流）`：在当前基本盘下，如何压榨出最大的经济收益。
* `版本 B（红利爆发流）`：如何利用下班时间低成本切入最新的时代风口。
* `版本 C（折中理想流）`：如果不考虑钱，如何把兴趣和生存做最合理的折中。

### 4. 严厉拦截盲目裸辞，手把手教你“副业过河”
你想裸辞去追寻梦想？AI 会直接开骂。它会强制为你定制一个 3-12 个月的最小可行性产品（MVP）测试包：把主业当掩护和提款机，利用业余时间把新赛道当成副业小样进行测试。**只有当副业收入稳了，它才允许你安全切换身份。**

## 🛠️ 底层引用的 5 大核心理论硬核内核

智能体的每一次毒舌点评和状态切换，都有以下 validated 框架在提供底层算法支撑：

1. **张雪峰现实主义（生存死磕线）**：评估任何路径的投资回报率（ROI），普通家庭以最快实现财务自给为唯一目标。
2. **斯坦福《设计人生》（奥德赛原型）**：打破唯一答案，强行设计 3 条并行的 5 年人生版本，提供探索勇气。
3. **塔勒布《反脆弱》（杠铃策略）**：拒绝高风险的中间状态，强制 80% 极度保守托底 + 20% 极度激进博红利。
4. **日本传统 Ikigai 模型（能量诊断）**：隐式审计“热爱、擅长、需要、给钱”的交集，严防纯功利主义导致的心理抑郁。
5. **伊瓦拉非线性职业转型（安全换赛道）**：严禁裸辞，主张“在行动中寻找新身份”，用副业进行无缝平滑跨行。

## 📊 开发者看这里：它是怎么“假装高级”的？（接口说明）

AI 本身不会画图，它不是网页前端。但这个 Skill 最出彩的地方在于，它在每一步对话后，**其 API 接口（`analysis` 节点）除了吐出毒舌的自然语言回复，还会同时吐出干净、标准的结构化 JSON 数据**（包含各项维度的 0-100 评分、平行宇宙的时间节点、杠铃分配比例）。

只要你把本接口接入前端，就可以一键驱动渲染出：
* 📊 **Ikigai 四象限雷达图**（一眼看出用户的盲目热情与残酷市场需求之间的落差）。
* ⚖️ **杠铃精力分配沙盘**（直观看到防守与进攻的资源占比）。
* 📅 **平行人生 5 年时间轴**（清晰标注前 3 个月、半年、5 年的每一步干啥）。
