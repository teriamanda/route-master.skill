# 🧭 Career-Life Designer Agent (人生战略规划与选择问询陪伴智能体)

> **拒绝灌鸡汤，拒绝死板问卷，拒绝大模型乱分析纯凭感觉。这是一个结合了“张雪峰式毒舌”与“设计人生方法论”的 9 步状态机聊天 AI 导师。它通过冷酷的代码框架限制大模型胡说八道，死卡你的真实筹码，强迫你面对现实，并吐出真正能落地防身的人生搞钱/选择路线图。**

---

## 🎯 它能实现什么？（扒开高大上，看具体干货）

这个项目在技术上不画大饼，本质上是一个基于 **FastAPI + 状态机控制流 + 极度严厉的结构化提示词** 的智能体内核。针对你面临的专业、考公、考研或转行迷茫，它切切实实能帮你做到以下 4 件事：

### 1. 像张雪峰一样逼你交出“生存底牌”
一般的 AI 聊着聊着就顺着你走了，这个 Agent 会在后台死死盯着你的记忆账本。在进行“人生观与工作观”反思时，它会踩刹车逼问你：*“对你来说什么最重要？绝对不能忍受什么？”* **如果不把价值观和现实底盘交代清楚，它会卡住流程，直到戳破你的认知幻觉。**

### 2. 将逃避式迷茫重构为“真问题”
很多人迷茫是因为问错了问题（例如盲目高喊“我要转行”）。本 Skill 引入**问题重构（Problem Framing）**机制，强行把你的焦虑转化为“如何能够找到既赚钱又有意义的事”等具体、可解的行动命题。

### 3. 强行推演 3 套同时存在的平行人生
拒绝单一路径焦虑。本 Skill 在 **Odyssey Planning（奥德赛计划）** 节点中，会强行逼你为自己设计 3 种完全不同的 5 年人生版本：
* `版本 A（稳妥前行流）`：基于现有资源，能走的最稳妥、最能搞钱的路。
* `版本 B（突发备选流）`：如果版本 A 走不通或发生意外，你的次优转向是什么。
* `版本 C（理想放飞流）`：在不考虑钱的前提下，你真正想做的是什么。

### 4. 严厉拦截盲目裸辞，手把手教你“原型试错”
你想裸辞去追寻梦想？AI 会直接开骂。在 **Prototyping（原型设计）** 阶段，它会强制为你定制 1 个月内的最小可行性行动包。通过做副业、找业内人访谈等低成本实验安全过河，行动胜于空想！

## 🛠️ 底层引用的 5 大核心理论硬核内核

智能体的每一次毒舌点评和状态切换，都有以下validated 框架在提供底层算法支撑：

1. **张雪峰现实主义（生存死磕线）**：评估任何路径的投资回报率（ROI），普通家庭以最快实现财务自给为唯一目标。
2. **斯坦福《设计人生》（奥德赛原型）**：本 Skill 的核心骨架。通过人生观/工作观、问题重构、奥德赛计划、原型设计，闭环打造探索勇气。
3. **塔勒布《反脆弱》（杠铃策略）**：拒绝高风险的中间状态，强制 80% 极度保守托底（主业保命） + 20% 极度激进博红利（副业翻盘）。
4. **日本传统 Ikigai 模型（能量诊断）**：隐式审计“热爱、擅长、需要、给钱”的交集，严防纯功利主义导致的心理抑郁。
5. **伊瓦拉非线性职业转型（安全换赛道）**：严禁裸辞，主张“在行动中寻找新身份”，用副业进行无缝平滑跨行。

## 🔄 9 步状态机聊天控制流

AI 会带着脑子陪你聊天，严格按照以下 9 个状态推进。**每一步它都会先对你的回答进行深度点评（通常是打破幻想），然后才会追问你下一个问题（收集底牌）**：
[ 状态 1: INITIAL (启动欢迎) ] ──> 融合5套硬核方法论破冰
│
▼
[ 状态 2: BACKGROUND (筹码审计) ] ──> 问: 家里能否支持？盘点最大痛点
│
▼
[ 状态 3: LIFE_VIEW (人生观反思) ] ──> 问: 80岁时想怎么度过？
│
▼
[ 状态 4: WORK_VIEW (工作观反思) ] ──> 问: 工作为了什么？绝对不能忍什么？
│
▼
[ 状态 5: PROBLEM_FRAMING (问题重构) ] ──> 转化迷茫，重构为“如何能够...”的真问题
│
▼
[ 状态 6: PASSION_SKILL & WAYFINDING (能量探索) ] ──> 盘点不用给钱也愿意干的事与特长
│
▼
[ 状态 7: MARKET_NEED (2026数据撞击) ] ──> 联网对撞行业残酷现实，评估风险承受度
│
▼
[ 状态 8: IKIGAI & ODYSSEY (平行人生) ] ──> 强推 3 条 5 年平行宇宙版本 (稳妥/备选/理想)
│
▼
[ 状态 9: BARBELL & TRANSITION (杠铃过河) ] ──> 拦截裸辞，强制 80/20 比例定制副业行动包
│
▼
[ 状态 10: FINAL_SUMMARY & COMPLETED ] ──> 整合输出完整的《人生设计蓝图》JSON

## 📊 开发者看这里：它是怎么“假装高级”的？（接口说明）

AI 本身不会画图，但这个 Skill 最出彩的地方在于，它在每一步对话后，**其 API 接口除了吐出毒舌的自然语言回复，还会同时吐出干净、标准的结构化 JSON 数据**（包含各项维度的 0-100 评分、平行宇宙的时间节点、杠铃分配比例）。

只要你把本接口接入前端，就可以一键驱动渲染出：
* 📊 **Ikigai 四象限雷达图**（一眼看出用户的盲目热情与残酷市场需求之间的落差）。
* ⚖️ **杠铃精力分配沙盘**（直观看到防守与进攻的资源占比）。
* 📅 **平行人生 5 年时间轴**（清晰标注前 3 个月、半年、5 年的每一步干啥）。
